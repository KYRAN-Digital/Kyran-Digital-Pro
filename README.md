# KYRAN Digital

Mobile-first business link page for KYRAN Digital.

## Files

- `index.html`: main page entry point
- `style.css`: page styling
- `logo.png`: logo/profile image used for favicon and brand lockup
- `contact.vcf`: downloadable contact card
- `assets/brand/`: logo and note image
- `assets/icons/`: branded quick-action and link icons
- `assets/reels/`: optimized video reels
- `assets/reels/posters/`: poster images for the reel cards

## Notes

- The `Website` action currently scrolls to the on-page showcase section because a final business domain was not provided.
- The `Google Maps` link uses a live search for `KYRAN Digital`. Replace it with a direct business pin when available.
- The original single-file draft, `kyran_digital_link_page.html`, is left in the project as a legacy reference.

## Deploy

This site is ready for static hosting such as GitHub Pages.
